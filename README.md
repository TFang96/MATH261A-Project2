# MATH261A-Project2: How stories and units inside a property affect how lot area impacts the property value. 
The second project for MATH261A. For this project we looked into the interaction effects that units/stories inside a building affects how lot area impacts assessed value. 

All files related to this project are in the folder, "Paper2".
Code File: r_code.R -- This file contains the data extraction, model fitting, and plot creation. It is included to show the workflow. All plots and tables are generated from within Quarto file.  
Quarto File: WriteUp.qmd
References.bib -- File containing citations. 
External LLM utilized to help plot residuals vs. fitted value plots, take subsets of data, and take a sample of our data.
